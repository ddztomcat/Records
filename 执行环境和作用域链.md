### 写在前面
相信大家都听说过execution context，翻译过来是执行上下文（执行环境），在操作系统中也有类似的上下文概念，它指得是存储在处理器各寄存器中的中间数据，还有context switchs（上下文切换）等，那js中的context跟os中的context相比，是不是也有异曲同工之妙呐？

>在仔细了解之前，先看个简单题目
```js
let a = 1
function jad() {
    let a = 2
    jdzw()
}
function jdzw() {
    console.log(a)
}
function jad1() {
    let a = 2
    function jdzw1() {
        console.log(a)
    }
    jdzw1()
}
jad()
jad1()
```
